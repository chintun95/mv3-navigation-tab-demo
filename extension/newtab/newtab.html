What just happened?

  1. This new tab page is an extension page declared with chrome_url_overrides.
  2. It sends TITLE_GET to the MV3 service worker.
  3. The service worker fetches a fake remote URL.
  4. A static Declarative Net Request rule redirects that request to the hosted server.
  5. The title is cached in chrome.storage.local so the worker can safely sleep.
